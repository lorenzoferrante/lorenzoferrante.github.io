Installazione (MacOS)

  • Cliccare sul bottone per scraricare il file Nachos.zip ed attendere la fine del download.
  • Il file verrà scaricato nella cartella Download. Quindi bisogna aprire la cartella, trovare il file Nachos.zip e fare doppio click su di esso per estrarlo.
  • Una volta estratto il file, comparirà una nuova app, chiamata Nachos. Per aprire l'app bisogna cliccare con il tasto detro sull'icona dell'app e scegliere Apri
  • Comparirà una finestra e bisognerà cliccare su Apri. A questo punto si aprirà l'app.
  • Si consiglia di spostare l'applicazione nella cartella Applicazioni, trascinando l'icona di Nachos nella cartella Applicazioni
